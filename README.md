- Matheus de Lara Silva - RM 572935

## Objetivo

Desenvolver um sistema de monitoramento para uma cápsula espacial utilizando Arduino e sensores.

## Componentes Utilizados

- Arduino Uno
- Sensor de temperatura
- Sensor de luminosidade (LDR)
- Botão para simulação de vibração
- Display LCD I2C

## Funcionalidades

- Leitura de temperatura
- Leitura de luminosidade
- Detecção de vibração
- Exibição dos dados em LCD

## Simulação

Projeto desenvolvido e testado na plataforma Wokwi.
